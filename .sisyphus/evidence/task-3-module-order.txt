**Sorting Algorithm**:
```typescript
function sortModuleFiles(files: string[]): string[] {
  return files.sort((a, b) => {
    const aMatch = a.match(/(\d+)-/);
    const bMatch = b.match(/(\d+)-/);
    
    if (!aMatch || !bMatch) {
      return a.localeCompare(b);
    }
    
    const aNum = parseInt(aMatch[1], 10);
    const bNum = parseInt(bMatch[1], 10);
    
    return aNum - bNum;
  });
}
```

**Key Features**:
- Regex extracts numeric prefix: /(\d+)-/
- Converts to integers for natural numeric sort
- Fallback to lexicographic sort if no match
- Handles edge cases (10-x comes after 02-x, not before)
